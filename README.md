# cross-platform-2048
Console based 2048 game for Linux, Windows, and DOS

To build this project:

-DOS:
  -Turbo C++ 1.01 or up
  >COMPILE.BAT


-Windows:

  -Mingw64 with gcc
  >gcc main.c game.c  hscr.c internal.c table.c -Wall


-Linux:

  -gcc
  >gcc main.c game.c  hscr.c internal.c table.c -Wall

#END
